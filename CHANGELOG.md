# 패치 이력

위젯이 하루 한 번 이 파일을 읽어 새 버전이 나왔는지 확인합니다.
맨 위 항목이 최신 버전이며, 형식은 `## v<버전> — <날짜>` 입니다.

## v2.7.0 — 2026-07-23

- 자동 업데이트: 새 버전이 나오면 알림이 뜨고, 트레이 메뉴에서 패치노트 확인 후 원클릭 설치
- 트레이 메뉴에 "패치 이력 보기" 추가

## v2.6.0 — 2026-07-23

- 활동 기반 폴링: 대화 중에는 60초, 유휴 시에는 10분 간격으로 조회해 서버 호출 한도(429)를 예방
- 429 발생 시 적응형 백오프 (Retry-After 헤더 존중)

## v2.5.0 — 2026-07-23

- 사용량 조회가 제한(429)돼도 답변 직후 신호로 재조회해 최신 값을 놓치지 않게 개선
