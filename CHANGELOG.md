# 패치 이력

위젯이 하루 한 번 이 파일을 읽어 새 버전이 나왔는지 확인합니다.
맨 위 항목이 최신 버전이며, 형식은 `## v<버전> — <날짜>` 입니다.

## v2.8.0 — 2026-07-24

- 토큰 자동 갱신: 만료된 토큰을 위젯이 스스로 갱신해 "토큰 만료" 문구와 매일 재로그인이 사라짐
  (죽은 설정 토큰은 자동 폐기 후 갱신 경로로 폴백, 다른 프로그램이 토큰을 회전시켜도 따라감,
  유휴 중에도 8시간마다 선제 갱신해 밤새 만료를 예방)
- 화면 캡처 중에 플로팅 바가 사라졌다 나타나던 문제 수정
  (Windows 캡처·Snipaste·카카오톡 캡처·알캡처 등 16종의 오버레이는 전체화면 가림으로 치지 않고,
  배경 재촬영은 색 변화가 1분 이상 유지될 때만 — 목록에 없는 도구가 바를 숨기면 로그에 이름이 남음)
- 서버 호출 제한(429) 시 Retry-After 대기를 전면 존중하고, 인증이 죽은 동안의 헛 재시도를 중단
